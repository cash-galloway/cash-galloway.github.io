The mistake here was that I took these restrictions too far on myself by deleting all my math functions that were fundamental to creating and coding an encryptor. I resolved this issue after trying to push through the next few class days and ended up resorting to classmates for assistance where they helped me realize that to even make the encryptor that I MUST use those deleted functions so I added them back and slowly ended up finding light of completing our encryptor. Another issue that arose during the development of our coding was creating the while loop that allowed users to select options. We knew how to start a while loop but not how to break one when needed to stop. Our code would run non-stop until the user forced paused the program or closed the tab corresponding to the running code. We resolved this issue by deciding to end the while loop by setting ‘x’ to false instead of typing “break” at the end of the loop.
