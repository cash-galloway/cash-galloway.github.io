WRITEUP FOR CHIKENOOF: ChikenOOF is now a finalized product with minor bugs. ChikenOOF not only was easy to make, but had some challenging obstacles to overcome along with my partner Jimmy Lembeck. For instance, the game was bound to choose a random integer between one through six, yet the game would only choose numbers two, four, and five. Jimmy and I spent numerous amounts of class time to somehow fix this properly coded number generator and ironically worked when we deleted the code and reapplied literally the same code. We both were dumbfounded by this emulator bug when the emulator has always been buggy. Not only was that little bug irritating, but it costs us valuable time to progress yet we focused down this issue instead. In addition, Jimmy and I both ran into another big conflict involving the brand logo when the app loads. First of all, the logo would never show up so we both had to spend a tremendous amount of time to resolve this. We eventually figured out this confusing resolution when we set the logo on the same screen as our game, but kept the image visable to have clock turn it invisible after a few seconds. Finally, I independently made our entire About screen. I figured out how to send the app into a different screen where I customized the entire screen when Jimmy had to miss a day of school. All in all, ChikenOOF is the best app I’ve ever invented as it was my first and it was my pleasure to experience this journey along with my man Jimmy as we both overcame struggles together to invent such a game.
VIDEO WRITEUP: When playing ChikenOOF, clicking “Play” plays alerts the user “Game On!!” which then sets a timer to spawn chickens randomly to click on for scoring. Then, when the user clicks “Reset”, it notifies along with a sound effect to notify the user “Game is paused and score is currently set to zero”. When this occurs, timer is now set off and will no longer spawn chickens until the user clicks “Play” again. When the user clicks the “About” button, the screen changes to talk about the founders of ChikenOOF, what school they attend, and to donate to our Paypal. It also shows our logo along with a button to return to the game whenever clicked.
